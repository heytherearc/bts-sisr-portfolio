XCP-ng / XOA Windows Server 2022 Virtualisation Routage / VLAN

Projet GSB

Mise en place d'une infrastructure complète basée sur l'hyperviseur XCP-ng et l'entreprise GSB. La virtualisation est ici le cœur du système, permettant de partitionner les ressources matérielles et de sécuriser les serveurs via Xen Orchestra (snapshots et sauvegardes).

Le réseau est segmenté par VLAN et sécurisé par des ACL pour isoler les flux critiques dès la couche physique.

L'ensemble est piloté par un domaine Active Directory sous Windows Server 2022, assurant la haute disponibilité des services DNS et DHCP.

Inventaire & Gestion de Parc

Réalisé durant mon stage à l'établissement Notre-Dame des Oiseaux (Paris 16), ce projet visait à déployer une solution complète de gestion de parc sur Debian 12.

J'ai d'abord installé OCS Inventory avant d'implémenter FusionInventory (GLPI-Agent) pour une remontée d'informations plus précise. Le déploiement incluait la configuration du serveur Apache, MariaDB et l'optimisation PHP.

L'enjeu majeur a été l'interfaçage avec l'Active Directory pour synchroniser et intégrer chaque device du domaine directement dans GLPI, permettant un suivi en temps réel de 100% de l'infrastructure.
