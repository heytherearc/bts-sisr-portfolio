Situation Professionnelle SISR

Infrastructure GSB

XCP-ng Windows Server 2022 VLAN 802.1Q Active Directory

01. Présentation du Projet

Dans le cadre du laboratoire GSB, j'ai été chargé de concevoir et déployer une infrastructure réseau robuste capable de supporter la segmentation des différents services de l'entreprise. L'objectif était d'assurer la haute disponibilité des services tout en garantissant une isolation stricte via des VLANs.

02. Tâches Techniques

Installation Bare Metal

Installation de l'hyperviseur XCP-ng sur serveur physique et configuration du management via Xen Orchestra.

Services Windows

Déploiement de deux contrôleurs de domaine (AD DS) avec réplication DNS et DHCP pour la gestion des clients GSB.

Configuration Switching

Configuration de ports de trunking (Tagged) et d'accès (Untagged) sur switch administrable pour supporter les VLANs.
